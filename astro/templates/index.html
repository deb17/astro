I, Debashish, am an astronomy enthusiast. It used to be my hobby during college days when I was part of an astronomy club. I have attended public talks, visited planetaria in India and abroad, and chased the solar eclipse. This is apart from reading books and watching sci-fi movies. This website is in memory of the good old days. It is built with the Python Flask framework and jQuery.
